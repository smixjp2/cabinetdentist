# **App Name**: DentiCare Maroc

## Core Features:

- Patient Management: Comprehensive patient profiles including personal information, medical history, allergies, and clinical notes.
- Appointment Scheduling & Reminders: Daily/weekly agenda, quick appointment booking, automated reminders via WhatsApp/SMS to reduce no-shows.
- Interactive Dental Chart: A digital dental chart to record treatments per tooth with a history log.
- Estimates & Billing: Create and manage treatment estimates, generate invoices, and track payments (paid, partial, unpaid).
- Simplified Accounting: Track daily/monthly revenue, income by treatment type, and identify most profitable patients, with Excel/PDF export.
- Inventory Management: Monitor consumable stock levels (gloves, anesthesia, etc.) with low stock alerts and usage history.
- Automated Patient Reminders via AI: Use a generative AI tool to personalize and send automated reminders for check-ups, cleanings, and follow-ups via WhatsApp with specific focus on appointment time.

## Style Guidelines:

- Primary color: Light teal (#A7DBD8), a calm and professional color evoking health and cleanliness.
- Background color: Very light gray (#F0F4F3), almost white, creating a clean and sterile backdrop.
- Accent color: Pale blue (#69A297), for calls to action and highlighting important information.
- Body and headline font: 'PT Sans', a clear and modern sans-serif font suitable for both body text and headlines.
- Use modern and simple icons to represent different treatments and patient information.
- Clean and intuitive layout with easy navigation, prioritizing quick access to essential features.
- Subtle animations for loading states and user feedback to enhance user experience.
